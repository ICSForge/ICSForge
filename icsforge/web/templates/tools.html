⚙ Allowed Networks

      

        Private IPs are already allowed — 10.x, 172.16–31.x, 192.168.x, loopback, and link-local work out of the box. You do not need to add anything for those.
      

      

        Some OT networks assign IP addresses from publicly-routed ranges that are only reachable inside the organisation — for example 130.75.0.0/24 or 195.10.20.0/24.
        Because these look like internet addresses, ICSForge blocks them by default. Add those ranges here so you can send scenarios to devices on that network.
